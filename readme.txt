== Description ==

This is the sample for generating pdf using php. You can edit and reuse the code for educational purposes only. Output contains some static content as well as dynamic also. You can use this for reference in your projects. You will be allowed to download the pdf file once form has been submitted.

** This is for demo only and nothing sanitization has been done. **

== Installation ==

Download zip or clone this file and unzipped it into your working forlder. Just run cca-form.php and you can see form to fill up for generating pdf. Some fields may or might not be working as for demo only. Once, you clicked button, pdf is generated and you can download that file.
